What does it do?
================

This extension is a part of TransitArt project. It enables easy publishing of public transport timetables. Data can be presented as timetables or journey planner. There is included demo data for the following cities:

* Auckland
* Budapest
* Chicago
* Łódz
* San Francisco
* Toronto
* Vancouver
* Warsaw

Contact us to publish own timetables data - office@transitart

According to map license terms this extension mustn’t be situated on not public website.
Note: this component has been designed as main element of the page. It is modifying site URL thus some conflicts with another extensions on the site may occur.
